Show Messages based on dates

== Description ==

This plugin is used to show messages based on dates. Start Date, End Date and Message section is added to Site Identity. Based on the dates input message will show.

=> Install Plugin and go to Appearance >> Customize >> Site Identity
=> Add Start Date, Expire Date and Second Tagline.
=> Based on the Start Date and Expired Date, Second Tagline message will show on site.
